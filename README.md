# A simple Arduino Oil Temp Gauge with Bluetooth Support
You need:
- Arduino Nano ESP32
- ADS1115 analog to digital converter for accurate readings
- GC9A01 Display
- a 10k NTC Thermistor (i used a Koso Oil Temperature Sensor)
- a 10k Resistor
- some Cables
